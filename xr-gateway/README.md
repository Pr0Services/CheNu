# CHE·NU XR Gateway

WebSocket-based XR rendering service for CHE·NU.

## Features
- XR Room Rendering (6 room types)
- Persona Encoding (PXR-1 & PXR-2)
- Unity/Unreal/WebXR compatibility
- Canonical geometry enforcement
- < 20ms latency target

## Endpoints
- `ws://localhost:8081/xr` - WebSocket connection
- `GET /xr/rooms` - List available rooms
- `POST /xr/render` - Render scene

## Safety Rules (LAWBOOK)
- All scenes are metaphorical
- All scenes are reversible
- No avatar autonomy
- No emotional manipulation
