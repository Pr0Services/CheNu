# CHE·NU INTERACTION PACK — CANONICAL (v1.0-Final)

> **Mode:** CHE·NU Interaction Mode Active  
> **Version:** 1.0-canonical  
> **Description:** Official interaction language of CHE·NU across 2D, 3D, XR, and hybrid environments.

---

## CORE PRINCIPLES

- Actions must be **reversible + traceable**
- Gestures mirror their meaning
- Movement must reduce cognitive load
- Agents are **summoned, never intrusive**
- Same interaction logic across 2D, 3D, XR
- User stays at the center of navigation
- Transitions feel physical and intentional
