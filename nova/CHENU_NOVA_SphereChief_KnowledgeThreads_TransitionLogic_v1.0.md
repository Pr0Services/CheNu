## 6) KNOWLEDGE THREADS — 3 TYPES ⚡

### THREAD 1 — TEMPORAL ⚡
> **Relie les événements dans le temps.**

| Example |
|---------|
| décision → meeting suivant → replay → résultat |

### THREAD 2 — SEMANTIC ⚡
> **Relie des concepts similaires.**

| Example |
|---------|
| "budget", "timeline", "prototype", "learning" |

### THREAD 3 — PROCEDURAL ⚡
> **Relie les étapes d'un processus.**

| Example |
